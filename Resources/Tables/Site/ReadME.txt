02.11.2025. Это сайт-редактор для создания новелл. 

Функционал: Вместе с основным кодом проекта позволяет создавать простые сцены: 

1.Создавать и удалять сцены, устанавливать их очередность.
2.Настраивать изображение, текст, музыку и звуки
3.Устанавливать анимации для всех, указанных выше элементов.
3.Добавлять персонажей, устанавливать анимации появления, исчезновения, перемещения и простоя(idle)
4.И самое главное: Отображать все изменения на экране редактирования.

Принцип работы проекта:

Каждая сцена поделена на 3 состояния: 
1.Начало сцены, 
2.Проигрывание сцены, 
3.Конец сцены. 

1.Начало сцены активируется при клике пользователя и запускает все нужные скрипты для активации нужной сцены. 
2.Проигрывание сцены поделено на 3 состояния(Их проигрывание можно пропустить, кликнув левой кнопкой мыши): Before, Default, After. Эти состояния привязаны к анимации изменения фона: До, по середине и после, соответственно.
3.Конец сцены активируется после того как отобразился весь текст, после чего все возвращается к 1 пункту.

Правила запуска сайта:

1.Сайт работает только в текущей иерархии файлов, для начала работы нужно нажать Enter и выбрать папку Tables, которая является родителем для текущей. Внутри нее нельзя менять уже вложенные файлы(Хотя можно добавлять новые и работать с ними)(Код полностью построен на текущей иерархии). Для сохранения изменений, нужно нажать кнопку Save all и опять выбрать папку Tables.

Заметки:

1. effect и animations - это окна для указания анимаций. Их можно указывать как Null для случаев когда они не нужны. Анимации указаны в скриптах проекта и соответствующих таблицах, их заполнение независимо друг от друга. Сайт считывает их из таблиц, по их названию
2. priority - Значение того, на какой фазе активируется анимация - Before, Default, After. Если анимация не null, то она имеет некоторое время для проигрывания, в частности, для музыки, это время равно времени изменения фона, так что рекомендуется ставить Before, в иных случаях, не сильно критично.
3. Save all - сохранить все изменения на всех сценах. Технически - записать все изменения в таблицы. Без нажатия этой кнопки, абсолютно любые изменения можно откатить.
4. Add Scene - добавить копию текущей сцены.
5. Delete Scene - удалить текущую сцену и персонажей
